## 비선형공식 엔진 모듈 구성

```
선순환 모듈 (A1:50% + B1:50%)
    ↓
멤버십 모듈 (x10 확장)
    ↓
5라운드 비선형 분배 [1,1,2,2,4]
    ↓
이탈모드 → 결합모드 (C1=4x, escape=50%)
    ↓
a→f 분배 체인 (a:50% → b:20% + c:12% + d:5% + e:1% + f:12%)
    ↓
광고주 적립 (소비지출의 5%)
    ↓
보정모드 (d:50M × 10회 → 150% → 120%)
    ↓
펀드존 (100%원금 + 20%증액 = 120%)
    ↓
최종 120% 적립
```

### 시뮬레이션 결과 (검증 완료)
- 10,000원 → 12,000P (120%) OK
- 10억원 → 12억원 (120%) OK
- A1+B1: 5억+5억 OK
- C1 풀: 40억 (4x) OK
- 이탈액: 5억 (50%) OK
- a→f 체인: 2억+1.2억+0.5억+0.1억+1.2억 = 5억 = a ✓
- 광고주적립: 5천만 (5%) OK
- 수수료(f): 1.2억 (12%) OK
- 보정값: 5천만 × 10회 = 5억 OK
- 펀드존 우선순위: bank(1) > insurance(2) > creditcard(3) > business(4) > consumer(5) OK
