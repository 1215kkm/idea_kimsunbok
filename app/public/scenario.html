핵심 개념: 다랜드에는 "가맹점"이 없다

다랜드는 모든 참여자가 사용자(소비자)입니다. 별도의 판매자 역할이나 가맹점 수수료가 존재하지 않습니다.

사용자가 신용카드로 어디서든 결제하면, 지출데이터 단말기가 영수증으로 지출을 증명하고, 본인 충전데이터에서 차감 후 비선형공식 알고리즘이 120% 증액 적립을 실행합니다.

그리고 같은 지출금액이 다른 멤버십 회원들에게도 전달되어, 그들도 본인 적립금에서 차감 후 비선형공식을 거쳐 120% 적립을 받습니다.

제1편. 일기 모드

박영희의 일기

3월 15일 (토) — 가입일

오늘 지은이가 자꾸 다랜드 다랜드 거려서 결국 깔았다.

"쓴 돈의 120%가 돌아온다"는데... 솔직히 믿기 힘들다. 세상에 공짜가 어딨어.

근데 가입하는데 30초밖에 안 걸렸다. 이름, 전화번호, 끝. 그리고 다랜드 내에 내 계좌가 생겼다. 여기에 포인트가 쌓이는 거고, 나중에 등록한 내 은행계좌로 출금할 수도 있다고 한다.

지은이가 말하길:

"가맹점 같은 거 없어. 네가 평소 쓰는 신용카드로 아무 데서나 결제하면 돼. 지출데이터 단말기가 네 영수증을 읽어서 자동으로 처리해줘. 네 충전데이터에서 지출금액이 차감되고, 비선형공식이 돌아가면서 120%로 증액 적립이 되는 거야."

아직 잘 모르겠지만 일단 깔아만 뒀다.

· · ·
3월 17일 (월) — 첫 결제일

출근길에 카페에서 아메리카노 4,000원. 평소처럼 신용카드로 결제했다. 지은이가 "다랜드 앱 켜놔봐!" 하길래 켜놨더니, 지출데이터 단말기가 영수증을 자동으로 인식했다.

[지출 확인] 카페 결제: 4,000원 (신용카드)
[데이터 차감] 충전데이터에서 -4,000 차감
[비선형공식] 분배 알고리즘 적용
[적립 완료] +4,800P → 다랜드 내 계좌에 적립

어... 4,000원 썼는데 4,800포인트가 다랜드 내 계좌에 들어왔다? 800원이 어디서 온 거지?

지은이한테 물어봤더니 이렇게 설명해줬다:

"네가 4,000원 결제하면, 지출데이터 단말기가 영수증으로 증명하고, 네 충전데이터에서 4,000이 차감돼. 그 다음에 비선형공식 알고리즘이 작동해서 120% 증액, 즉 4,800P가 적립되는 거야. 그리고 여기서 끝이 아니야. 네 지출금액 4,000원이 다른 멤버십 회원들에게도 전달돼. 그 회원들도 본인 적립금에서 차감 → 비선형공식 → 120% 적립을 받아. 서로가 서로에게 이득을 주는 구조야."

음... 아직 좀 헷갈리지만 일단 800원 이득인 건 맞는 것 같다.

점심 — 두 번째 결제

점심에 김치찌개 8,000원도 평소처럼 신용카드로 결제. 다랜드 앱이 자동 인식했다.

[지출 확인] 식당 결제: 8,000원 (신용카드)
[데이터 차감] 충전데이터에서 -8,000 차감
[비선형공식] 분배 알고리즘 적용
[적립 완료] +9,600P (120% 증액 적립)

오늘 하루만 12,000원 쓰고 14,400P 받았다. 2,400원 이득.

그리고 잠깐, 앱에 알림이 하나 더 왔다.

[멤버십 분배] 다른 회원 이지은 님의 지출이 분배되었습니다.
[적립금 차감] 내 적립금에서 -5,000 차감
[비선형공식] 분배 알고리즘 적용
[적립 완료] +6,000P (120% 증액 적립)

지은이가 어딘가에서 5,000원을 썼는데, 그 지출이 나한테도 전달돼서 내 적립금에서 차감된 후 120%로 돌아왔다. 1,000P 이득. 이게 멤버십 분배 구조구나.

· · ·
3월 20일 (목) — 포인트로 결제

쌓인 포인트가 꽤 됐다. 편의점에서 우유 + 빵 3,500원짜리를 포인트로 결제했다.

[지출 확인] 편의점 결제: 3,500P (포인트 사용)
[데이터 차감] 충전데이터에서 -3,500 차감
[비선형공식] 분배 알고리즘 적용
[적립 완료] +4,200P (120% 증액 적립)

미쳤다. 포인트로 결제했는데 또 120% 적립이 된다. 3,500P 쓰고 4,200P 받았으니 700P 이득.

그리고 이 결제도 다른 멤버십 회원들에게 분배된다. 내가 쓸 때마다 다른 사람도 이득을 보고, 다른 사람이 쓸 때마다 나도 이득을 본다.

· · ·
3월 20일 (목) — 드디어 이해했다

앱에 있는 "엔진 설명" 페이지를 정독했다. 드디어 이해했다.

전체 흐름: 사용자 결제 → 120% 적립 → 계좌 적립 → 출금

1단계. 사용자가 신용카드로 어디서든 결제한다.

2단계. 다랜드가 지출데이터를 인식(지출데이터 단말기 영수증 증명)한다.

3단계. 비선형공식 알고리즘이 작동하여 120% 증액된다.

4단계. 증액된 포인트가 다랜드 내 본인 계좌에 적립된다.

5단계. 다랜드에 등록한 본인 은행계좌로 출금할 수 있다. (1P = 1원)

6단계. 동시에, 지출금액이 다른 멤버십 회원들에게 전달된다.

7단계. 전달받은 회원들도 비선형공식 → 120% 적립 → 각자의 다랜드 계좌에 적립.

시스템 풀의 재원 — 돈이 들어오는 곳

1. 광고주 광고비: 신한은행 같은 기업이 다랜드 앱에 광고를 싣고 월 10억 지불. 사용자가 광고를 시청하면 보상(포인트)을 받고, 이 광고비가 시스템 풀에 들어간다.

2. 프리미엄 멤버십: 월 9,900원 구독자는 적립률 130%로 업그레이드. 이 구독료도 풀에 들어간다.

3. 멤버십 분배 순환: 사용자 간 지출금액이 계속 순환하면서 풀이 자연스럽게 확장된다.

왜 유지가 되는가

광고주가 넣는 돈 + 프리미엄 구독료 + 멤버십 순환 효과 = 모든 사용자의 120% 증액 적립 재원

풀에 쌓이는 돈 > 나가는 돈 → 시스템 유지

풀에 쌓이는 돈 < 나가는 돈 → 보정모드가 120%→110%로 자동 조절

핵심: 내 주머니에서 추가로 나오는 게 아니라, 광고주 주머니에서 나오는 거였다.

네이버 검색할 때 광고 보이잖아. 그 광고비가 네이버 매출이 되듯이, 다랜드에선 그 광고비가 나한테 오는 거다.

그리고 나의 소비가 다른 회원들에게도 이득을 주고, 다른 회원들의 소비가 나에게도 이득을 준다. 모두가 서로에게 120%를 만들어주는 구조.

어차피 쓸 생활비에서 매달 15만원 이상의 보너스!

지은이의 리포트도 봤다. 지은이는 나보다 더 오래 썼고 회원도 많이 연결되어 있어서, 멤버십 분배로 받는 적립이 더 컸다.

그리고 드디어 출금을 해봤다. 다랜드 앱에서 "출금하기" 버튼을 눌러서 등록해둔 내 은행계좌(신한은행)로 100,000P를 출금 신청했다.

[출금 신청] 다랜드 내 계좌 → 신한은행 ***-****-1234
[출금 금액] 100,000P = 100,000원 (1P = 1원)
[수수료] 무료
[입금 예정] 영업일 1~2일 내

다음 날 통장을 확인하니 진짜 10만원이 들어와 있었다. 이건 그냥 숫자가 아니라 진짜 돈이었다. 다랜드 내 계좌에 쌓인 포인트가 현금으로 변한 순간이었다.

이지은의 일기

2월 1일 (토) — 가입 2개월 차

다랜드를 쓴 지 두 달이 됐다. 처음엔 나도 반신반의했는데, 이제 확실히 이해했다.

내가 카페에서 5,000원을 신용카드로 결제하면:

[내 결제] 카페 5,000원 → 충전데이터 차감 → 비선형공식 → +6,000P (120%)
[멤버십 분배] 내 지출 5,000원이 연결된 회원들에게 전달
  → 회원A: 적립금 차감 후 비선형공식 → +6,000P (120%)
  → 회원B: 적립금 차감 후 비선형공식 → +6,000P (120%)
  → 회원C: 적립금 차감 후 비선형공식 → +6,000P (120%)

내 한 번의 소비가 나 포함 여러 명에게 120%를 만들어준다. 그리고 다른 사람이 소비할 때마다 나도 분배를 받는다.

효율이 4.3배. 왜?

게다가 포인트를 받은 사용자가 결제하면 → 본인도 120% 적립 → 다른 회원들도 120% 적립 → 회원들이 또 사용 → 사용자 증가 → 광고 노출 증가 → 선순환.

광고주 관점의 핵심 차이

기존 플랫폼: 광고비 → 플랫폼이 독식 → 소비자에게 돌아가는 것 없음

다랜드: 광고비 → 멤버십 풀에 유입 → 사용자들의 120% 적립 재원 → 사용자 충성도 극대화 → 광고 효율 상승

제2편. 소설 모드 — 「항아리 속의 물」

1장. 새어나가는 돈

박영희는 월급날이 가장 슬픈 사람이었다.

250만원이 통장에 찍히는 순간, 자동이체가 시작된다. 월세 55만원. 통신비 7만원. 보험 12만원. 남은 176만원으로 한 달을 산다.

커피 한 잔, 점심 한 끼, 버스비, 편의점 간식. 하나하나는 작은데 모이면 거대했다. 월말이면 통장엔 늘 2~3만원만 남았다.

"돈이 자꾸 새어나가."

영희는 가계부 앱을 깔았다 지웠다를 반복했다. 기록해봤자 달라지는 건 없었다. 쓸 돈은 써야 하니까.

2장. "쓴 돈이 돌아온다고?"

"영희야, 이거 써봐."

동료 이지은이 핸드폰을 들이밀었다. 보라색 앱 아이콘. 다랜드.

"뭔데?"

"결제하면 120%가 포인트로 돌아와. 그것도 나만 받는 게 아니라, 내가 쓸 때마다 다른 회원들도 120%를 받아."

"그런 게 어딨어. 무슨 사기 아니야?"

지은이 웃었다.

"나도 처음엔 그랬어. 근데 생각해봐. 네이버 쓸 때 광고 보이잖아. 그 광고비가 네이버 매출이 되잖아. 다랜드는 그 광고비가 우리한테 오는 거야. 기업이 광고비를 내면, 그게 멤버십 풀에 쌓이고, 우리가 결제할 때마다 그 풀이 비선형공식으로 작동해서 120%가 되는 거야.

그리고 가맹점 같은 거 없어. 네 평소 신용카드로 아무 데서나 결제하면 지출데이터 단말기가 영수증을 읽어서 자동 처리해줘."

"...진짜?"

"내가 지난 달에 42만원 썼는데 본인 적립만 50만 4천 포인트 받았어. 거기에 다른 회원들 분배까지 합치면 60만 포인트 넘어. 이거 편의점에서도 쓸 수 있어."

영희는 반신반의하며 앱을 깔았다.

다음 날 아침. 출근길.

삐.

그런데 이번엔 다랜드 앱에서도 알림이 울렸다. 지출데이터 단말기가 영수증 데이터를 자동으로 읽어들인 것이다.

4장. 멤버십의 힘

영희가 다랜드를 쓴 지 3일째 되던 날, 앱에 알림이 연달아 왔다.

[멤버십 분배] 회원 이지은 님의 지출이 분배되었습니다.
  내 적립금 차감: -12,000 → 비선형공식 → +14,400P (120%)

[멤버십 분배] 회원 김민수 님의 지출이 분배되었습니다.
  내 적립금 차감: -8,000 → 비선형공식 → +9,600P (120%)

[멤버십 분배] 회원 정수진 님의 지출이 분배되었습니다.
  내 적립금 차감: -5,500 → 비선형공식 → +6,600P (120%)

영희는 눈을 비볐다. 자기가 아무것도 안 했는데 적립금이 늘어나고 있었다.

다른 회원들이 각자의 일상에서 소비를 할 때마다, 그 지출금액이 멤버십을 통해 영희에게도 전달된 것이다. 영희의 적립금에서 차감은 되지만, 비선형공식이 120%로 증액해서 돌려주니까 차감된 것보다 더 많이 돌아온다.

이게 멤버십의 힘이었다.

회원이 많아질수록, 소비가 활발해질수록, 분배의 횟수가 늘어나고, 모두의 120% 적립이 빈번해진다. 한 사람의 소비가 사라지지 않고 네트워크 전체를 돌며 모두에게 120%를 만든다.

5장. 돈은 어디서 오는가

영희는 그날 밤 침대에서 앱을 들여다봤다. "엔진 설명" 페이지를 눌렀다. 하나하나 단계를 넘기면서 읽었다.

Q. 120%가 가능한 이유?

돈이 하늘에서 떨어지는 게 아닙니다.

1. 광고주의 광고비
신한은행, 삼성전자 같은 기업이 다랜드 앱에 광고를 싣습니다. 사용자가 광고를 시청하면 보상(포인트)을 받습니다. 이 광고비가 시스템 풀에 들어갑니다.

2. 프리미엄 구독료
월 9,900원 프리미엄 회원은 적립률 130%. 이 구독료도 풀에 들어갑니다.

3. 멤버십 순환 효과
한 사람의 소비가 다른 회원들에게 전달되고, 그 회원들의 소비가 또 다른 회원들에게 전달됩니다. 이 순환이 반복될수록 풀이 자연스럽게 확장됩니다. 비선형공식이 이 순환의 효율을 극대화합니다.

이것들이 멤버십 풀을 만들고, 비선형공식이 모든 사용자에게 120%로 분배합니다.

"항아리 속의 물 총량은 같다. 바가지만 바뀔 뿐."

영희는 드디어 이해했다.

네이버를 쓸 때, 광고를 보는 대가로 네이버가 돈을 벌었다. 유튜브를 볼 때, 광고를 보는 대가로 구글이 돈을 벌었다.

다랜드는 그 구조를 뒤집은 것이다. 광고를 보는 대가를 플랫폼이 아니라 사용자가 가져간다. 그리고 사용자의 소비가 다른 사용자들에게까지 120%의 이득을 만들어준다.

7장. 선순환

그렇게 반년이 흘렀다.

영희 → 카페에서 신용카드 결제 → 영희 본인: 충전데이터 차감 → 비선형공식 → 120% 적립 → 회원 지은: 적립금 차감 → 비선형공식 → 120% 적립 → 회원 민수: 적립금 차감 → 비선형공식 → 120% 적립 지은 → 식당에서 신용카드 결제 → 지은 본인: 충전데이터 차감 → 비선형공식 → 120% 적립 → 회원 영희: 적립금 차감 → 비선형공식 → 120% 적립 → 회원 수진: 적립금 차감 → 비선형공식 → 120% 적립 한 사람의 소비가 사라지지 않고 멤버십 네트워크 전체를 돌며 모두에게 120%로 돌아갔다.

신한은행은 다랜드 광고로 카드회원을 8,500명 모았다. 기존 방식보다 4배 효율적이었다. 광고비를 늘렸다.

광고비가 늘어나자 멤버십 풀이 커졌다. 풀이 커지자 120% 적립의 재원이 풍부해졌다. 적립이 풍부해지자 사용자가 몰렸다. 사용자가 몰리자 멤버십 분배가 더 활발해졌다. 분배가 활발해지자 모두의 이득이 커졌다. 이득이 커지자 광고주가 더 투자했다.

눈덩이였다.

마지막 장. 항아리

영희는 1년이 지나서야 다랜드가 만든 세상을 완전히 이해했다.

기존 세상

소비자가 돈을 쓴다 → 플랫폼이 가져간다 → 끝

소비자에게 돌아오는 것은 없다. 돈은 일방통행으로 사라진다.

다랜드 세상

사용자A가 신용카드로 결제한다

→ 다랜드가 지출데이터를 인식한다

→ 비선형공식이 120%로 증액한다

→ 다랜드 내 A의 계좌에 120% 적립된다

→ A는 등록된 본인 은행계좌로 출금할 수 있다 (1P = 1원)

→ 동시에 같은 지출금액이 다른 회원들에게 전달된다

→ 회원B, C, D도 각자 비선형공식 → 120% → 각자의 다랜드 계좌에 적립

→ 회원B가 결제하면 A에게도 분배 → A도 120% 적립

→ 이 순환이 끝없이 반복된다

돈의 총량은 같다. 다만, 기존에는 플랫폼 한 곳이 독식했던 것을 모든 사용자가 서로에게 120%를 만들어주는 구조로 바꿨을 뿐이다.
